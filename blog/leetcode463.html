Leetcode 463. Island Perimeter

Question:

You are given a map in form of a two-dimensional integer grid where 1 represents land and 0 represents water. Grid cells are connected horizontally/vertically (not diagonally). The grid is completely surrounded by water, and there is exactly one island (i.e., one or more connected land cells). The island doesn't have "lakes" (water inside that isn't connected to the water around the island). One cell is a square with side length 1. The grid is rectangular, width and height don't exceed 100. Determine the perimeter of the island.

You may assume the greed factor is always positive.

You cannot assign more than one cookie to one child.

Example 1:
Input:

[[0,1,0,0], [1,1,1,0], [0,1,0,0], [1,1,0,0]]

Output:

16

Explanation:

The perimeter is the 16 yellow stripes in the image below:

Solution:

Traverse the matrix, find all elements with value equals to 1. And count the number of neghibors which equals to 0 or is the boundary of the maxtrix. Summing all the selected elements and return the result.

Code: 

class Solution {
public:
    int islandPerimeter(vector>& grid) {
        int res = 0;
        for(int i=0; i < grid.size();i++){
            for(int j=0; j < grid[i].size(); j++){
                if(grid[i][j] == 1){
                    int temp_b = 4;
                    if(i>0 && grid[i-1][j] == 1) temp_b--;
                    if(j>0 && grid[i][j-1] == 1) temp_b--;
                    if(i < grid.size()-1 && grid[i+1][j] == 1) temp_b--;
                    if(j < grid[1].size()-1 && grid[i][j+1] == 1) temp_b--;
                    res += temp_b;
                }
            }
        }
        return res;
    }
};
