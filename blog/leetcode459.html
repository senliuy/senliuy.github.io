Leetcode 459. Repeated Substring Pattern

Question:

Given a non-empty string check if it can be constructed by taking a substring of it and appending multiple copies of the substring together. You may assume the given string consists of lowercase English letters only and its length will not exceed 10000.

Example 1:
Input:

"abab"

Output:

True

Explanation:

It's the substring "ab" twice.

Example 2:
Input:

"aba"

Output:

False

Example 3:
Input:

"abcabcabcabc"

Output:

True

Explanation:

It's the substring "abc" four times. (And the substring "abcabc" twice.)

Solution:

Using brutal force can slove this problem but its too time consuming. However, the KMS algorithm prvoided with a solution with O(n+m) time complexity.

Using KMP algorithm finding a vector, if this string (length = n) has repeated strings, the len%(len-next[n-1]) == 0 && next[n-1]!=0.

Code: 

class Solution {
public:
    bool repeatedSubstringPattern(string str) {
        int len = str.size();
        vector < int > next(len, 0);
        int k=0;
        for(int i=1;i < len;i++){
            while(k>0 && str[i] != str[k]) k = next[k-1];
            if( str[i] == str[k] ) next[i]=++k;
        }
        return next[len-1]!=0 && len % (len - next[len-1]) == 0;
    }
};
